Vježba Git & Github

Vaš zadatak je kreirati lokalni repozitorij te unutar njega primijeniti sustav za verzioniranje koda Git. Kreirati ćete jednu HTML datoteku te u njoj objaviti određeni sadržaj na temu po vašem odabiru.Također, kreirati ćete i jednu CSS datoteku te preko nje dodijeliti osnovne stilove HTML sadržaju (postavite nekoliko CSS pravila). U konačnici, lokalni repozitorij ćete objaviti na GitHub servisu.
